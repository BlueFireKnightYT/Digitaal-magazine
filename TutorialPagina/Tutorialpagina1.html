Gebruikte prompt:
maak een tutorial over de basis van HTML

Elke HTML-pagina begint met dezelfde basis. Bovenaan staat < !DOCTYPE html >, waarmee je de browser laat weten dat het om HTML5 gaat. De hele pagina staat binnen < html >...< /html >. Het < head >-gedeelte bevat informatie die de gebruiker niet ziet, zoals de titel van de pagina of instellingen zoals de tekencodering. In het < body >-gedeelte staat juist alles wat wél zichtbaar wordt, zoals tekst, koppen, afbeeldingen en links. Als je tekst op een pagina wilt zetten, gebruik je paragrafen met < p > en koppen met < h1 > t/m < h6 >. < h1 > is de grootste en belangrijkste kop, < h6 > de kleinste. Je kunt tekst vet maken met < b > of < strong >, en cursief met < i > of < em >. Links maak je met de < a >-tag. Die werkt met een href-attribuut dat aangeeft naar welke website je verwijst. Afbeeldingen voeg je toe met < img >. Die tag heeft geen sluittag nodig. Je geeft een src op voor het bestand en een alt-tekst die verschijnt als de afbeelding niet kan geladen worden. Voor opsommingen gebruik je lijsten. Een ongeordende lijst (met bolletjes) wordt gemaakt met < ul >, en een geordende lijst (met nummers) met < ol >. Elk lijstpunt maak je met < li >. Daarnaast bestaan er nog de elementen < div > en < span >. Een < div > is een blok-element dat vaak gebruikt wordt om grote stukken van een pagina te groeperen, terwijl < span > juist een klein stukje tekst markeert binnen een regel. Deze worden vooral belangrijk wanneer je later CSS gaat gebruiken om de pagina op te maken. Tot slot kun je commentaar toevoegen in HTML met < !-- ... -- >. Dit wordt niet getoond op de pagina, maar is handig voor aantekeningen. Een heel eenvoudige HTML-pagina bevat dus een doctype, een < html >-deel, een < head > met de titel, en een < body > met zichtbare inhoud zoals tekst, links, lijsten en afbeeldingen. Wanneer je deze elementen begrijpt, kun je al je eerste eenvoudige webpagina’s maken.

Conclusie:
Een heel eenvoudige HTML-pagina bevat dus een doctype, een < html >-deel, een < head > met de titel, en een < body > met zichtbare inhoud zoals tekst, links, lijsten en afbeeldingen. Wanneer je deze elementen begrijpt, kun je al je eerste eenvoudige webpagina’s maken.
